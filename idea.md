# CareerOS – Proof-Based Career Engine

## The Problem: The Loop of Stagnation

Tier-2/3 students face:
- Tutorial Hell
- Choice Paralysis
- Disconnected progress (GitHub, LeetCode, learning scattered)
- No bridge between learning and job readiness

They consume content but fail to produce proof.

---

## The Solution: CareerOS

CareerOS acts as a Career Operating System.

It:
- Converts vague goals into structured milestones
- Forces proof-of-work (GitHub commits, problem solving)
- Tracks readiness score
- Provides visible career progress

---

## Core Innovation

Automated Proof-of-Work Verification Engine.

Instead of tracking “hours studied”,
we track:
- GitHub commits
- Project repositories
- DSA problem count
- Deployment proof

---

## Target Users

- Tier-2 / Tier-3 engineering students
- Self-taught developers
- Career switchers

---

## MVP Features

1. GitHub OAuth Login
2. Goal Selection (Frontend / Backend / DSA)
3. Milestone Tracking
4. Daily Task Card
5. Readiness Score Dashboard
6. Streak System
